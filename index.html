TMDB-5000

O TMDB-5000 é um conjunto de dados do The Movie Database (TMDB) disponibilizado pelo Kaggle. São 4804 filmes extraídos do TMDB e atualizados pela última vez em 2017, contando com aproximadamente 9MB.

Nesta visualização é possível explorar esses dados e verificar algumas questões sobre os filmes. Para objetivos de análise o conjunto de dados foi tratado, unindo os arquivos do Kaggle em uma única planilha e removendo 77 filmes que não tinham nenhuma nota, não haviam sido lançados na época que o dataset foi publicado ou não tinham dados de produtora, elenco ou equipe. Após o tratamento, o conjunto de dados conta com 4727 filmes. Além disso, apenas as 5 primeiras entradas de elenco e equipe são consideradas.

A resolução mínima de monitor recomendada para visualizar os gráficos é de 1366x768.
